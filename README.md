# Alloweb

**Kids allowance app** - complete family tasks, get allowance money and purchase items from your wishlist.

## Built With

- Angular 6 & Material Design
- MySQL Cloud Database & Sequelize
- Nodejs & Expressjs
- Ebay API
- Multer
